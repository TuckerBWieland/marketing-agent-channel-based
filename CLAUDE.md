## Cross-Channel Conventions

### UTM Parameters
All links in marketing content must include UTM parameters:
```
?utm_source=[channel]&utm_medium=[type]&utm_campaign=[campaign-name]
```

| Parameter | Convention | Example |
|-----------|-----------|---------|
| utm_source | Channel name, lowercase | `email`, `linkedin`, `google` |
| utm_medium | Content type | `newsletter`, `social`, `cpc`, `blog` |
| utm_campaign | Campaign slug, kebab-case | `q1-product-launch`, `webinar-series-2024` |

### Campaign Naming
All campaigns follow: `[YYYY-QN]-[channel]-[campaign-name]`
Example: `2024-Q2-email-spring-launch`
